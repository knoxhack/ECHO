# Structure Templates

To generate the actual NBT structure template for the Pre-Gridfall Research Vault:

1. Build the vault in-game using the RelicTech blocks.
2. Use the structure block or `/place template` to save it.
3. Export the NBT to `data/echorelictech/structure/pre_gridfall_research_vault/start.nbt`.

Until a proper template is generated, the worldgen JSON references an empty placeholder
and the structure will not spawn naturally. Use `/place structure echorelictech:pre_gridfall_research_vault`
to test placement once the NBT is available.
