# Convoy Vehicle Reference Notes

These screenshots are visual references for the vehicle model and texture pass only. Do not bake screenshot UI, callout labels, health bars, hotbars, or chat text into runtime assets.

- `scrap_bike_reference.png`: compact rideable field bike with separate thrust pods, thin handlebar silhouette, compact seat block, and bright cyan vent panels.
- `wasteland_rover_front.png` / `wasteland_rover_side.png`: green armored scouting rover with deep cab frame, layered hood/fenders, separate wheel modules, roof antenna cluster, cyan headlights, and varied 16px camo breakup.
- `cargo_crawler_front.png` / `cargo_crawler_side.png`: long cargo crawler with front cab depth, lower track modules, cage/slat cargo frame, stacked cargo crates, dusty olive/brown palette, and rear utility block.
- `armored_relay_truck_front.png` / `armored_relay_truck_side.png`: heavy grey relay truck with cab armor depth, layered bumper, side module depth, roof comms stack, relay hardware, and cyan dashboard/power panels.
- `all_vehicle_scale_reference.png`: scale and silhouette reference for all four vehicles together.
