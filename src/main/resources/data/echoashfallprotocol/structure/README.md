# Structure NBT and Underground Guardian Path

The required root NBT files now exist in this folder and remain the compatibility anchor for `/locate`, datapack structure discovery, and Signal Scanner POI routing:

- `bio_lab.nbt`
- `data_center_ruin.nbt`
- `drop_pod.nbt`
- `military_vault.nbt`
- `reactor_ruin.nbt`

Gameplay dungeons are generated by `ProceduralStructureGenerator`. The procedural system is the source of truth for navigable rooms, loot chests, hazards, underground guardian arenas, and biome boss placement.

## Biome Guardian Structures

Each wasteland biome has a required guardian profile. The visible surface structure is only the entrance; the real site is a buried Gridfall control node 18-36 blocks below the surface with a themed descent route, side rooms, hazard pockets, defenders, loot, and one clear boss chamber.

| Biome | Surface entrance | Underground arena | Guardian |
|---|---|---|---|
| `the_wasteland` | Half-buried rescue hatch | Buried Rescue-Control Lab | Wasteland Sentinel |
| `ruined_plains` | Reinforced bunker door | Underground Remnant Command Bunker | Plains Warlord |
| `ruined_cityscape` | Subway service stair | Subway Data-Vault Ambush Site | City Ruin Stalker |
| `industrial_ruins` | Ruined freight lift | Factory Sublevel Maintenance Core | Industrial Juggernaut |
| `toxic_swamp` | Toxic sinkhole | Sunken Bio-Hive | Toxic Hive Matriarch |
| `crash_zone_wasteland` | Impact vault breach | Buried Military Crash Vault | Crash Zone Colossus |
| `radiation_zone` | Reactor access hatch | Reactor Containment Basement | Radiation Behemoth |
| `cryogenic_ruins` | Frozen service shaft | Frozen Cryo-Storage Facility | Cryogenic Overseer |
| `nexus_scar` | Folded Nexus breach | Folded Anomaly Chamber | Nexus Scar Avatar |

Drop pods are side structures only and are not used as boss arenas.

The procedural generator gives every guardian a distinct visual theme: rescue-control concrete for the Wasteland Sentinel, fortified bunker staging for the Plains Warlord, subway/sculk cues for the City Stalker, industrial lift hardware for the Juggernaut, toxic growth for the Hive Matriarch, crash debris for the Colossus, containment lighting for the Behemoth, ice storage for the Overseer, and Nexus anomaly anchors for the Scar Avatar. Each theme still preserves a clear scanner entrance, descent route, and reserved boss-spawn zone.

## Debug Commands

Use these commands as the primary test path for guardian entrance and dungeon generation:

```mcfunction
/genpoi biome_main <biome>
/genpoi procedural <type>
```

Examples:

```mcfunction
/genpoi biome_main toxic_swamp
/genpoi biome_main the_wasteland
/genpoi procedural reactor_ruin
```

For a full guardian visual pass, smoke-test all biome-main profiles:

```mcfunction
/genpoi biome_main the_wasteland
/genpoi biome_main ruined_plains
/genpoi biome_main ruined_cityscape
/genpoi biome_main industrial_ruins
/genpoi biome_main toxic_swamp
/genpoi biome_main crash_zone_wasteland
/genpoi biome_main radiation_zone
/genpoi biome_main cryogenic_ruins
/genpoi biome_main nexus_scar
```

The biome-main command selects the canonical guardian profile for the biome, places a visible surface entrance, carves an access route, generates the underground arena, stores the entrance/arena position in guardian site data, and spawns its guardian once in a valid boss room.

## Loot Tables

Boss-arc structures use the prepared chest loot tables under `data/echoashfallprotocol/loot_table/chests/`. Their reward flow includes filters, RadAway, thermal support, dense alloy, power cells, schematics, Nexus materials, and data logs such as `data_log_biome_bosses`.
